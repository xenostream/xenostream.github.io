XME [ Xeno Markdown Editor ]

먼저 마크다운(Markdown)에 대한 사항은 건너띄고 시작하겠습니다.

http://plan9.tistory.com/14  <== 마크다운 문법 가이드를 참고 하시기 바랍니다.  

간단히, 일반텍스트에 HTML과 같은 최소한의 서식을 적용할 수 있는 문서화 시스템을 마크다운 문서라고 부르며,
의미는 없지만 확장자 .md 나 .markdown 으로 끝나는 문서를 뜻합니다.

마크다운으로 문서를 작성 할려면 메모장이나 전문 편집기등으로 작성할 수 있습니다. 하지만 결론적으로는
일반 텍스트형태로 저장된 마크다운 문서를 웹브라우저에서 서식이 적용된 상태로 보여주어야할 시스템이 필요합니다.

github, wordpress, php markdown, wiki... 등의 요즘 대부분의 블로그나 위키에서는 마크다운 문법을 문서를 작성
할 수 있도록 하고있습니다. 

필자의 경우에도, 대부분을 워드패드로 작성후 PDF로 보관 했었습니다.
하지만 네이버 블로그나 티스토리와 같은 곳에서 문서를 게시할 때는 자체 에디터나 HTML 형태로 변환해서 다시
편집해야하만! 합니다. 

온라인, 오프라인에 수많은 마크다운 에디터가 존재합니다. 각각 장단점이 존재합니다만, 필자의 성격상 
작고, 빨라야하고, 실행파일 한개만으로 구성되어 있어야하는 것을 추구하는 터라서 본 *Xeno Markdown Editor*를 
만들었습니다. (가장 필요한 최소한의 기능만으로 구성되어 있습니다)

단지 실행파일 한개로 구성되며, 497KB 크기입니다.

필자의 경우 현재 대부분의 문서를 마크다운으로 작성하고 있습니다. 간단한 문법으로 최대한 원문 텍스트를 보존하는
서식화 도구이기 때문이며, 호환성이 갈수록 높아지고 있습니다. (표준마크다운 + 변종 마크다운들……)

Xeno Markdown Editor는 다음과 같은 특징을 가지고 있습니다. 

1. 기본 인코딩은 UTF-8 입니다.(UTF-8)
     한글, 영문등을 원활하기 표기하기 위한 코드는 유니코드만한것이 없으므로, UTF-8 인코딩으로 처리합니다.
 
  2. 실행파일 한개로 구성되어 있습니다. (Single File)
     xme.exe 497KB 의 단일파일로 구성되어 있습니다. 설치과정도 필요없으며 단지 실행하면 그만입니다. 
     윈도우 탐색기에서 .md 파일을 더블클릭한 후 xme.exe 만 지정하면 언제든지 마크다운을 편집할 수 있습니다. 
 
  3. 문서 작성시 빈번한 이미지/파일 삽입은 드래그드롭으로 간단히 처리됩니다.(Drag & Drop)
    편집중인 마크다운 파일이 존재하는 폴더에 이미지파일은 "_images/날짜", 첨부파일은 "_attachements/파일명"  
     형태로 저장되며, 단지 드래그 드롭으로 처리하면 됩니다. 마크다운 링크 태그로 자동 변환됩니다. (폴더 자동생성)  
    마크다운 문서 파일이 존재하는 폴더에 "_images, _attatchements"폴더로 생성됩니다.   
   첨부이미지나 파일이 없다면 당연히 생성하지 않습니다. 파일단위로 이미지나 첨부파일을 관리하기 편해집니다.    
 
  4. 이미지나 파일을 클립보드에 복사한 후 곧바로 붙여넣기 할 수 있습니다.( Paste from Clipboard)
     상기와 마찬가지로 클립보드에 이미지가 복사되었다면 "./_images" 폴더에, 파일이라면  
      "./_attatchements" 폴더에 붙여넣기를 하는 순간 곧바로 저장됩니다. 
 
  5. 마크다운의 프리뷰화면의 소스코드를 곧바로 클립보드로 내보낼 수 있습니다.(Export to Clipboard)
     먼저 마크다운으로 작성한 후, 프리뷰화면으로 모습을 살펴본 후, github나 tumblr와 같은 실제 웹으로 게시할 때는  
     대부분 HTML형태로 변환해서 저장합니다. Xeno Markdown Editor에서 작성한 문서는 버튼만 클릭하면 곧바로   
     작성중인 문서의 프리뷰화면에 대한 HTML 태그를 클립보드로 복사합니다. (그냥 붙여넣기로 사용하시면 됩니다)  
 
  6. 대부분의 문법을 기본적으로 지원합니다.
     표준 마크다운 문법과 깃허브, 워드프레스, PHP 마크다운등 대부분의 문법에 대한 프리뷰를 지원합니다

상기의 용도 외에도 여러가지 용도로 사용 합니다. 개인 용도로 만든 프로그램이라서, 어디 내놓기도 민망하지만… 필요하실 분이 있을 거라 생각 합니다.
실행파일 하나로 구성되어 있습니다. 필자의 좁은 판단으로, 사용할 것이라 생각하는 기능들만 넣었습니다.
  첨부파일을 다운로드 하신 후 원하는 폴더나 Windows 폴더에 넣고 그냥 사용하시면 됩니다. 

Xeno Markdown Editor Download - x64

사용법은 너무도 직관적이라 어렵지 않습니다.  메뉴도 몇가지 없습니다. 화면의 좌측이 마크다운 소스이고, 우측편이 프리뷰화면 입니다.  메뉴에 단축키도 적용되어 있습니다. 
좌측하단의 오렌지색박스 부분은 “Copy preview html code to Clipboard”기능입니다. 

항상 최후버전의 파일을 첨부하오니 사용하시기 바랍니다. 

감사합니다.
