Go by Example: [01] Hello World

Posted on Feb 16, 2016
#gobyexample

Hello World

첫 번째예제는 화면에 Hello world메세지를 출력하는 예제입니다. 다음은 코드입니다…

package main

import "fmt"

func main() {
    fmt.Println("Hello world")
}

프로그램을 실행하려면, 상기의 코드를 hello-world.go라는 파일명으로 저장한 후,
(프로그램이름 지정은 강제사항이 아닙니다.알기쉬운 이름으로 저장하시기 바랍니다.)
go run 명령을 입력하면 됩니다. 다음과 같이 입력하시기 바랍니다.

######$표시는 프롬프트입니다. 곧바로 실행 결과를 표시하고 있습니다. 참고하시기 바랍니다.

$go run hello-world.go
hello world

상기의 예제는 곧바로 실행한 경우입니다. 대부분의 경우는 실행파일을 생성하기 위해서,
프로그램을 작성하는 경우일 것입니다. 이제 다음의 명령을(build) 입력해서 실행파일을 생성합니다.

$ go build hello-world.go
hello-world    hello-world.go

실행파일(hello-world)이 생성되었으니, 곧바로 실행해 보겠습니다.

$ ./hello-world
Hello world

이제, Go언어를 이용해서, 프로그램을 작성하는 법과 컴파일해서 실행파일을 생성하는 법과,
프로그램을 실행하는 방법에 대해서 익혔습니다. 모든것은 이단계를 벗어나지 않습니다.

계속 Go언어의 여러가지를 알아 보도록 하겠습니다.
