권필진의 개인적인 블로그 입니다.

본 블로그는 별도의 메뉴가 없습니다. 먼저 사이트 상단의 아이콘으로 조작합니다.

먼저 상단 그림에서 3개의 아이콘이 존재합니다.

순서상으로 about, tags, archive 입니다.

  • about 아이콘은 현재 페이지 입니다.

  • tags 아이콘은 일반적인 메뉴에 해당합니다.(문서마다 단일 태그로 작성되어 있습니다.)
    예를 들어, devtools 2로 표시되면 개발도구 관련 문서가 2개 있다는 의미입니다.

  • archive 아이콘은 전체 글목록 입니다. 작성일 기준으로 정렬 됩니다.

본 사이트는 Go언어 기반의 Hugo Static Site Generator로 GitHub's Pages에서 운영됩니다.
