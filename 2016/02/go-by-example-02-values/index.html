Go by Example: [02] Values

Posted on Feb 16, 2016
#gobyexample

Values

Go언어는 다양한 데이터값을 다룰수 있습니다. 문자열, 정수형, 실수형, 불 등을 지원합니다.
몇가지 예를 통해서 알아보겠습니다.

package main

import "fmt"

func main() {
    fmt.Println("go" + "lang")             // 문자열
    
    fmt.Println("1 + 1 = ", 1 + 1)         // 정수 
    ftm.Println("7.0 / 3.0 = ", 7.0 / 3.0) // 실수 
    
    fmt.Println(true && false)             // 참거짓(불)
    fmt.Println(true || false)
    fmt.Println(!true)
}

문자열은 + 기호를 통해서 연결할 수 있습니다. 단지 데이터값만 보시기 바랍니다.

$ go run values.go
golang
1 + 1 = 2
7.0 / 3.0 = 2.3333333333333335
false
true
false

실행하면 상기와 같습니다.
